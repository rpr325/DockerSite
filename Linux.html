System Information

  • 'uname -a'

    Display Linux system information.

  • 'uname -r'

    Display kernel release information.

  • 'uptime'

    Display how long the system has been running with load uptime.

  • 'whoami'

    Display user logged in.

  • 'last reboot'

    Display system reboot history.

Hardware Information

  • 'cat /proc/cpuinfo'

    Display CPU information.

  • 'cat /proc/meminfo'

    Display memory information.

  • 'dmesg'

    Display messages in kernel ring buffer.

  • 'free -h'

    Display free and used memory ( -h for human readable, -m for MB, -g for GB.).

  • 'lsusb -tv'

    Display USB devices.

Performance Monitoring and Statistics

  • 'top'

    Display and manage the top processes.

  • 'htop'

    Interactive process viewer (top alternative).

  • 'mpstat 1'

    Display processor related statistics.

  • 'vmstat 1'

    Display virtual memory statistics.

  • 'iostat 1'

    Display I/O statistics.

User Information and Management

  • 'id'

    Display the user and group ids of your current user.

  • 'last'

    Display the last users who have logged onto the system.

  • 'who'

    Show who is logged into the system.

  • 'w'

    Show who is logged in and what they are doing.

  • 'groupadd test'

    DCreate a group named "test".

File and Directory Commands

  • 'ls -al'

    List all files in a long listing (detailed) format

  • 'pwd'

    Display the present working directory.

  • 'mkdir directory'

    Create a directory.

  • 'rm file'

    Remove (delete) file.

  • 'rm -r directory'

    Remove the directory and its contents recursively.

Process Management

  • 'ps'

    Display your currently running processes.

  • 'ps -ef'

    Display all the currently running processes on the system.

  • 'ps -ef | grep processname'

    Display process information for processname.

  • 'kill pid'

    Kill process with process ID of pid.

  • 'killall processname'

    Kill all processes named processname.

File Permissions

  • 'chmod 777 filename'

    User - rwx Group - rwx World - rwx

  • 'chmod 775 filename'

    User - rwx Group - rwx World - r-x

  • 'chmod 755 filename'

    User - rwx Group - r-x World - r-x

  • 'chmod 664 filename'

    User - rw- Group - rw- World - r--

  • 'chmod 644 filename'

    User - rw- Group - r-- World - r--

Networking

  • 'ethtool eth0'

    Query or control network driver and hardware settings.

  • 'ping host'

    Send ICMP echo request to host.

  • 'whois domain'

    Display whois information for domain.

  • 'dig domain'

    Display DNS information for domain.

  • 'dig -x IP_ADDRESS'

    Reverse lookup of IP_ADDRESS.

Archives (TAR Files)

  • 'tar cf archive.tar directory'

    Create tar named archive.tar containing directory.

  • 'tar xf archive.tar'

    Extract the contents from archive.tar.

  • 'tar czf archive.tar.gz directory'

    Create a gzip compressed tar file name archive.tar.gz.

  • 'tar xzf archive.tar.gz'

    DExtract a gzip compressed tar file.

  • 'tar cjf archive.tar.bz2 directory'

    Create a tar file with bzip2 compression.
