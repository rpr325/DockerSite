Web Server Setup



1. Install Docker on Ubuntu

Start a terminal session and input the curl -fsSL https://get.docker.com | sh command to install Docker.

Next use the systemctl start docker and systemctl status docker to start the Docker service.

Then execute with the docker command

2. Setting Up the Apache Container

Add the Apache container and map between host and container. Input sudo docker run -dit --name site-web -p 8080:80 -v /home/user/website/:/usr/local/apache2/htdocs/ httpd:2.4.

To confirm the Apache operation with the sudo docker ps command.

Create the first webpage (index.html) inside the host directory with the following command vi /home/website/docker.html

Add HTML to the webpage

Point your browser to the local host IP address

Finally stop the continer with the sudo docker stop site-web
